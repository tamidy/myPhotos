I couldn't find a solution online. What I did was create a whole new project with the command line:

npx create-react-app <appname> --template typescript

then I copied over my work into the new project by deleting individual matching files and 
pasting them over to the new project and it works great. This command line allows you to just focus 
on modifying the app without stopping the app and running tsc to convert the tsx to js.

If you do this I would suggest just adding the include to tsconfig.json and not deleting that file.

I followed this video from Udemy to get the command line: 

https://cognizant.udemy.com/course/react-and-typescript-build-a-portfolio-project/learn/lecture/24208850#overview

If you don't want to do this, cd to src, type tsc then you should be able to type npm start. 
However, if you make any changes to the tsx files, you have to stop the app and type tsc again then 
npm start again.
